# Greek and Roman Mythology

Returns JSON data about humans, places, and God(desse)s that appear in the works of Virgil, by id, name, and work they appear in.

- URL
  /list/:id  
  /list/name/:name  
  /list/work/:work  

- Method  
`Get`  

- URL Params  
Required:  
```
id = string  
name = string  
work = string
```
